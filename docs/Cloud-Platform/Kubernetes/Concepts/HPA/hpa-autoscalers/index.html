Requirementsâ€‹

  • External metrics provider (k8s 1.10+)
  • Prometheus
  • Prometheus adapter

Deploymentâ€‹

Make sure Deployment gathers prometheus metrics:

  template:
    metadata:
      annotations:
        prometheus.io/path: "/metrics"
        prometheus.io/scrape: "true"
        prometheus.io/port: "8000"

Prometheus Adapterâ€‹

Add a custom rule on prometheus or a Configmap for the adapter config and mount it as volume on the prometheus adapter deployment:

  • configmap
apiVersion: v1
kind: ConfigMap
metadata:
  name: adapter-config
  namespace: monitoring
data:
  config.yaml: |
    rules:
    - seriesQuery: 'nginx_vts_server_requests_total'
      resources:
        overrides:
          kubernetes_namespace:
            resource: namespace
          kubernetes_pod_name:
            resource: pod
      name:
        matches: "^(.*)_total"
        as: "${1}_per_second"
      metricsQuery: (sum(rate(<<.Series>>{<<.LabelMatchers>>}[1m])) by (<<.GroupBy>>))
  • prometheus adapter deployment
        volumeMounts:
        - mountPath: /etc/adapter/
          name: config
          readOnly: true
      volumes:
      - name: config
        configMap:
          name: adapter-config
