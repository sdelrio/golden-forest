# Duplicati

Backup solution for workstations. Store securily encrypted backups in the cloud.

## OS

* Windows
* macOS
* Debian/Ubuntu
* Fedora/Redhat
* Synology

## [Backends](https://duplicati.readthedocs.io/en/latest/01-introduction/#supported-backends)


* FTP
* SSH
* WebDav
* Backblaze B2
* Tardigrade
* OneDrive
* Amazon S3
* Google Drive
* box.com
* Mega
* hubiC
