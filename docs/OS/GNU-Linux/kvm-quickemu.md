# KVM/Quickemu

Quickly create and run optimised Windows, macOS and Linux desktop virtual machines. 

## Install

### Ubuntu

```bash
sudo apt-add-repository ppa:flexiondotorg/quickemu
sudo apt update
sudo apt install quickemu
```
