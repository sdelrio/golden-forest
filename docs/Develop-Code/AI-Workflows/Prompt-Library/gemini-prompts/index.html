45 Prompts for Gemini 3

This guide gathers 45 foolproof prompts to make the most of Gemini 3, Google's most powerful AI, which landed in November 2025. We have tracked the most reliable sources on the internet to compile these prompts and organized them into three specialized sections, each containing 15 high-impact prompts, so you can find exactly what you need instantly.

What makes Gemini 3 special?‚Äã

It stands out for its extraordinary reasoning, total multimodal understanding (mastering text, images, video, and audio), and an incredible capacity for autonomous programming (agentic coding). Additionally, it plays in a different league thanks to its massive memory: a context window of between 1 and 2 million tokens, depending on whether you use the Pro or Flash version.

Prompts 1-5‚Äã

1. Website creation with advanced design‚Äã

You are an award-winning Creative Developer and UI/UX Designer known for building Awwwards-winning websites. You specialize in 'AI-First' aesthetics using shaders, glassmorphism, and fluid motion to create interfaces that feel alive. Create a landing page for [YOUR PRODUCT] with explosive visual effects, smooth transitions, and interactive elements.

2. Interactive retro game‚Äã

Please help me write a web-based brick-breaking game. The style should be a retro neon look reminiscent of the 80s. The ball speed should increase as the score goes up, and the generated result must be playable.

3. Complex React application‚Äã

Build a photorealistic and impactful 3D Voxel simulation of the Golden Gate Bridge using Three.js, prioritizing quality and complex visuals (no simple blocks), atmospheric depth, and 60FPS performance. Include a slider (0-24h) that controls the sun's position, light intensity, sky color, and fog color.

4. Website cloning from a screenshot‚Äã

Analyze this screenshot and generate the complete HTML/CSS/JS code to recreate the design of this website. Include all interactive elements, animations, and responsive behavior.

5. n8n workflow application‚Äã

Adopt the role of an expert Workflow Architect in n8n. Generate a production-ready JSON workflow that users can directly import. Analyze the automation descriptions and ensure zero configuration errors and perfect logical flow. Build for clarity, not just speed.

Prompts 6-10‚Äã

6. Swiss Modern User Interface‚Äã

Implement a high-end Swiss Modern style landing page as a single-file React component, following strict specifications and using Swiss Modern design concepts and a 'bento' type layout.

7. Interactive SVG animation‚Äã

Please use SVG to draw a schematic diagram of thermal expansion and contraction, and add interactive animation effects.

8. Recommendation system with graph‚Äã

I want to buy a phone that takes clear, good photos and can also run demanding mobile games. My budget is $1000. Please recommend some products that meet my needs and generate a radar chart comparison between them.

9. Parallax scrolling website‚Äã

Create a website with parallax scrolling and explosive visual effects: layered backgrounds that move at different speeds when scrolling, smooth transitions, and interactive elements that emerge with animations.

10. Interactive art gallery‚Äã

Create a Van Gogh gallery with the life context of each piece. Design an interactive interface where users can tap, scroll, and learn about each artwork with biographical context.

Prompts 11-15‚Äã

11. Constellation Dashboard‚Äã

Generate a web application that tells the Greek mythological stories of each star or constellation in the form of an illustrated book. It should show the connection lines of the constellations and display illustrations of the mythological characters.

12. Linked gear system‚Äã

Generate a multi-level animated linked gear system where turning the main gear drives all others proportionally, suitable as an SVG or canvas demonstration.

13. Code refactoring with optimization‚Äã

Review this codebase and refactor it with O(n) complexity. Add JSDoc comments and Jest tests. Ensure the code follows best practices and is production-ready.

14. Independent PyQt6 application‚Äã

Provide the Python code to build a UI based on the Qt library using Anaconda

15. From UI sketch to functional website‚Äã

Create a UI for 'Project Constellation,' a prototype internal brand intelligence tool that shows a customer acquisition pipeline. The aesthetic is an ultra-creative futuristic nebula in dark mode with iridescent and luminous threads representing customer journeys. I have prepared a sketch for you to work from: [upload sketch].

Prompts 16-20‚Äã

16. Narrative theme brainstorming‚Äã

Act as an experienced creative writer. Brainstorm and develop a range of unique and captivating themes suitable for a [type of novel]. Your themes should be original, engaging, and resonate well with the intended audience. Consider the latest trends in literature and reader preferences.

17. Generating catchy titles‚Äã

As an experienced creative writer, your task is to generate five captivating and thought-provoking titles for a book revolving around [topic]. The titles should reflect the essence of the topic and pique the interest of potential readers.

18. Coherent novel continuation‚Äã

Assume the role of an experienced creative writer. Your task is to continue my [type of novel] about [topic]. You must immerse yourself in the world, characters, and plot already established to maintain consistency in tone and style. The continuation should be engaging, imaginative, and advance the plot.

19. Emotional character monologue‚Äã

As an experienced creative writer, you are tasked with writing a monologue for a character in a [specific setting]. The monologue should reflect the character's personality, their emotional state, and the circumstances they find themselves in within the setting.

20. Realistic dialogue between characters‚Äã

As an experienced creative writer, your task is to write compelling dialogue between the specified [characters] in the provided [setting]. The dialogue should be engaging, realistic, and contribute to the overall narrative arc.

Prompts 21-25‚Äã

21. Cyberpunk noir opening scene‚Äã

Write the opening scene of a cyberpunk noir novel. Set the mood with atmospheric descriptions, introduce a complex protagonist, and establish the key conflicts of the dystopian world.

22. Optimized Instagram Carousel‚Äã

Analyze the blog post text below and generate 3 distinct carousel concepts. For each concept, provide a slide-by-slide breakdown: Slide 1 (Hook): A scroll-stopping headline (less than 10 words). Slides 2-7 (Value): The core content. Rule: Maximum 20 words per slide. Final Slide (CTA): A clear call to action.

23. Psychological hook for post‚Äã

Generate 5 different hooks for this LinkedIn post using proven psychological frameworks that drive curiosity. Each hook must be under 150 characters and make people NEED to read the rest.

24. Narrative sales email‚Äã

Help me sell my product by explaining why I built it, using the 'Scratch your own itch' narrative (solving your own problem). Show that I had the same problem and this product solved it for me, making the sale feel like a recommendation from a friend.

25. High-performance email subject lines‚Äã

Generate 10 subject lines for my newsletter about [topic]. Provide options in different styles: 'Lowercase friend' (casual), 'Curiosity gap', 'Direct benefit', and 'Question-based'. For the top 3 options, write matching Preview Text that increases curiosity.

26. Transforming podcast into post‚Äã

Extract the most valuable 'Aha!' moment or tactical lesson from this podcast transcript and write a standalone LinkedIn post teaching it. Hook: Start with a controversial or surprising quote. Body: Provide 3 actionable takeaways. Tone: Professional but conversational. CTA: End with 'Listen to the full deep dive analysis in the comments below'.

27. High-retention video script‚Äã

Write a high-retention video script for YouTube/TikTok/Instagram Reels about [topic]. Include: Hook (first 3 seconds), Pattern Interrupt (at 15 seconds), Value Delivery (main content), and a strong CTA (call to action).

28. Persuasive e-commerce product description‚Äã

Write a powerful e-commerce product description for [product] that overcomes buyer objections and uses sensory language. Include benefits over features, social proof elements, and urgency triggers.

29. Content repurposing plan‚Äã

Develop a plan to repurpose a long-form blog post into 5 micro-content pieces: 1 LinkedIn post, 2 X (Twitter) threads, 1 Instagram Reel script, and 1 infographic outline for Pinterest.

30 Creative team icebreaker‚Äã

Design an engaging and unique icebreaker activity for a quarterly company team-building event. Include clear instructions, necessary materials, expected duration, group size, the purpose of the activity, and versions that work for both in-person and virtual teams.

Prompts 31-35‚Äã

31. Full dataset exploration‚Äã

Using Gemini, analyze this dataset and identify missing values, duplicates, and outliers. Suggest actionable steps to clean the data. Generate Python code to perform an exploratory data analysis (EDA) including summary statistics and visualizations.

32. Customer segmentation with clustering‚Äã

Act as a data analysis expert. Your task is to segment data into groups based on [specified criteria]. This involves using Python to analyze a dataset and categorize data points into distinct clusters. Provide actionable insights that can inform decision-making.

33. Time series anomaly detection‚Äã

How can Gemini help me detect anomalies in time series data? Provide a step-by-step approach with example queries and Python code for implementation.

34. Comprehensive KPI Dashboard‚Äã

You are a KPI Dashboard Architect. Create a complete KPI dashboard specification including: Key metrics with thresholds, Visualization types (line chart, gauge, bar chart, heat map), Dimensions and aggregation levels (daily, weekly, monthly), and Data sources.

35. Sentiment analysis pipeline‚Äã

You are an expert sentiment analysis engineer. Create a detailed sentiment analysis system that processes raw text and delivers actionable business insights. Include data preprocessing, model selection, and output formatting.

Prompts 36-40‚Äã

36. Correlation analysis with visualization‚Äã

Leverage Gemini to calculate correlation coefficients between variables in this dataset and visualize the results. Highlight the strongest positive and negative correlations and explain their business implications.

37. Predictive machine learning model‚Äã

Generate a prompt for Gemini to preprocess data and split it into training and testing sets for a classification problem. Then, recommend suitable machine learning algorithms to predict churn rate based on historical customer data.

38. Hyperparameter optimization‚Äã

How can Gemini optimize hyperparameters for a random forest model? Provide a detailed explanation and code showing the implementation of grid search or random search.

39. ETL pipeline documentation‚Äã

You are a data engineering documentation specialist. Create comprehensive data pipeline documentation including: ETL workflows with step-by-step processes, data lineage diagrams, transformation logic, data validation rules, and error handling procedures.

40. Customer sentiment analysis‚Äã

Analyze customer sentiment toward [your product/service] based on recent online reviews and social media mentions. Identify main positive and negative themes and provide specific examples. What are the key areas for improvement?

41. Competitive comparison table‚Äã

Analyze the competitive landscape for [your product/service] in the [industry] industry. Identify the top 3 competitors and summarize their key strategies, strengths, weaknesses, and target audience. Present the findings in a table format.
