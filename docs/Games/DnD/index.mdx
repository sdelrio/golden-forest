# D&D

This repository centralizes Dungeons & Dragons house custom rules, balance patches, and essential mechanics. It serves as a comprehensive reference to ensure consistency and streamline tabletop gameplay.

The following resources are categorized to streamline character creation and gameplay:

| Category | Related Content |
| :--- | :--- |
| **Classes** | Character archetypes custom balance and fixes for homebrew|
| **Weapons & Armor** | Combat equipment, defensive gear custom balance and fixes|
| **Magic** | Spellcasting rules, spell lists, and enchanted artifacts custom balance and fixes |
| **Injuries** | Injuries treatment and related mechanics |
