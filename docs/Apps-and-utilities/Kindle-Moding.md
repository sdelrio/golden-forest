# Kindle Modding

Modifying a Kindle device allows you to unlock additional features, install third-party applications, and remove restrictions imposed by Amazon. This guide covers the process for jailbreaking a Kindle Voyage (7th generation) running firmware version 5.13.7.

## Prerequisites

Before proceeding, ensure you have:

- A Kindle Voyage (Model B013, 7th generation, 2014)
- Firmware version 5.13.7 installed
- A USB cable for connecting to your computer
- Access to a computer with internet connectivity

## Jailbreak Method: WinterBreak

[WinterBreak](https://kindlemodding.org/jailbreaking/WinterBreak/) is the recommended exploit for Kindle devices running firmware 5.x. This method takes advantage of a vulnerability in the Kindle's webkit browser.

### Step 1: Install WinterBreak

1. Download the WinterBreak package from the [Kindle Modding homepage](https://kindlemodding.org/jailbreaking/index.html)
2. Connect your Kindle to your computer via USB
3. Copy the update file to your Kindle's root directory
4. On your Kindle, navigate to **Settings** > **Update Your Kindle**
5. Wait for the installation to complete

### Step 2: Apply HotFix

After jailbreaking, install the [HotFix](https://kindlemodding.org/jailbreaking/post-jailbreak/setting-up-a-hotfix/) to maintain access and prevent the jailbreak from being removed during normal device operation.

### Step 3: Install Kual and MRPI

[Kual](https://kindlemodding.org/jailbreaking/post-jailbreak/installing-kual-mrpi/) is a launcher application that provides a menu for accessing installed extensions. MRPI (MR Package Installer) enables the installation of additional packages.

1. Download Kual and MRPI from the linked guide
2. Install both packages following the provided instructions
3. Access Kual from your Kindle's home screen

### Step 4: Disable OTA Updates

[Disable automatic over-the-air updates](https://kindlemodding.org/jailbreaking/post-jailbreak/disable-ota) to prevent Amazon from automatically updating your firmware, which could remove the jailbreak.

### Step 5: Re-enable the Store

If you disabled the Amazon Store during the jailbreak process, follow the [re-enabling guide](https://kindlemodding.org/jailbreaking/post-jailbreak/re-enabling-the-store/) to restore access to purchasing books.

### Step 6: Install KOReader

[KOReader](https://kindlemodding.org/jailbreaking/post-jailbreak/koreader) is a versatile e-book reader application that supports more file formats than the native Kindle app, including EPUB, PDF, and DJVU. It also offers advanced reading features like custom dictionaries and superior PDF handling.

## Identifying Your Kindle Model

To identify your specific Kindle model, refer to the following resources:

- [Kindle Models Reference](https://kindlemodding.org/kindle-models.html) — Comprehensive list of all Kindle models
- [Amazon Forum (Spanish)](https://es.amazonforum.com/s/identify-your-kindle-e-reader) — Alternative method to identify your device

The model number (such as B013 for the Voyage) is printed on the back of your device, or you can find it in your Amazon account settings.
