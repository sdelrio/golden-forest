# Personal Finance Tools

This document covers free and open-source tools for managing personal finances, including budgeting and investment portfolio tracking.

## Firefly III

[Firefly III](https://firefly-iii.org/) is a free and open-source personal finance manager designed to help you track and analyze your spending habits.

### Features

- **Budget Management**: Create and track budgets across multiple categories
- **Account Management**: Support for multiple bank accounts, credit cards, and cash accounts
- **Transaction Tracking**: Detailed recording of income and expenses with categorization
- **Reports and Charts**: Visual insights into spending patterns, income vs expenses, and budget adherence
- **Data Import**: Import transactions from bank CSV exports
- **Automation**: Rule-based transaction categorization and recurring transactions
- **Self-Hosted**: Full control over your data with self-hosting capabilities

### Resources

- [Homepage](https://firefly-iii.org/)
- [GitHub](https://github.com/firefly-iii/firefly-iii)
- [Documentation](https://docs.firefly-iii.org/)

### OS Compatibility

- **Self-Hosted**: Linux, macOS, Windows (via Docker)
- **Web Interface**: Any modern browser (access when self-hosted)

## Portfolio Performance

[Portfolio Performance](https://www.portfolio-performance.info) is a free, open-source portfolio management tool that helps you track and analyze your investments across multiple accounts and asset classes.

### Features

- **Portfolio Tracking**: Monitor your entire investment portfolio in one place
- **Performance Analysis**: Calculate and visualize portfolio performance over time
- **Asset Support**: Supports stocks, bonds, funds, ETFs, cryptocurrencies, and more
- **Transaction Management**: Record buys, sells, dividends, and other investment transactions
- **Reporting**: Generate detailed reports on holdings, allocations, and performance metrics
- **Watchlists**: Track securities you don't currently own
- **Data Import**: Import transactions from various brokers and formats
- **Cross-Platform**: Available for Windows, macOS, and Linux
- **Privacy-Focused**: All data stays on your device; no cloud account required

### OS Compatibility

- Windows, macOS, Linux

### Resources

- [Homepage](https://www.portfolio-performance.info)
- [GitHub](https://github.com/portfolio-performance/portfolio-performance)
- [Documentation](https://help.portfolio-performance.info/)

## Actual Budget

[Actual Budget](https://actualbudget.org/) is a privacy-focused, local-first personal finance application that uses the envelope budgeting method. Originally a commercial product, it was open-sourced and now offers robust budgeting capabilities for self-hosting.

### Features

- **Envelope Budgeting**: Assign money to categories (envelopes) and track spending against each
- **Local-First**: All data stored locally on your device; no mandatory cloud sync
- **Multi-Device Sync**: Optional end-to-end encrypted sync across devices
- **Bank Synchronization**: Supports connections via SimpleFIN and Plaid (with extensions)
- **Reports**: Visual reports showing spending patterns and budget progress
- **Rule-Based Automation**: Automate transaction categorization with custom rules
- **CSV Import**: Import transactions from bank exports
- **Self-Hosted**: Deploy your own instance using Docker or fly.io
- **Cross-Platform**: Web-based interface accessible from any browser

### OS Compatibility

- **Hosted/App**: Any modern browser (Windows, macOS, Linux, iOS, Android)
- **Self-Hosted**: Linux, macOS, Windows (via Docker)

### Resources

- [Homepage](https://actualbudget.org/)
- [GitHub](https://github.com/actualbudget/actual)
- [Documentation](https://actualbudget.org/docs/)

## Money Manager Ex

[Money Manager Ex](https://moneymanagerex.org/) (MMEX) is a free, open-source, cross-platform personal finance application that has been in active development for over 15 years. It provides comprehensive tools for tracking income, expenses, and assets.

### Features

- **Double-Entry Accounting**: Professional accounting method for accurate financial tracking
- **Account Management**: Support for multiple accounts including checking, savings, credit cards, and investments
- **Transaction Recording**: Detailed transaction logging with categories, payees, and notes
- **Budget Tracking**: Create and monitor budgets across categories
- **Reports**: Generate financial reports including income/expense summaries, account summaries, and cash flow analysis
- **Data Import**: Import from CSV, QIF, and OFX formats
- **Data Export**: Export to CSV, HTML, and PDF formats
- **Recurring Transactions**: Automate recurring income and expenses
- **Cross-Platform**: Available for Windows, Linux, and macOS
- **Portable Mode**: Can run from USB drive without installation

### OS Compatibility

- Windows, macOS, Linux
- Portable mode: Runs from USB on Windows

### Resources

- [Homepage](https://moneymanagerex.org/)
- [GitHub](https://github.com/moneymanagerex/moneymanagerex)
- [Documentation](https://moneymanagerex.org/docs/)
