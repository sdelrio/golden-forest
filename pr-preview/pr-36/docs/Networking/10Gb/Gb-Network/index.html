10 GB Network at home

Basically try to go fiber instead of cable.

2016​

  • 10GB Home Part 1 - Introuction
  • 10GB Home Network Part 2 - Peer to Peer
  • 10GB Home Network Part 3 - Build a Switch

Hardware​

  • MB:Supermicro X10SRH-CLN4F
  • CPU: Xeon E5 Quad Core
  • CPU Fan (Narrow ILM) Noctua i4
  • Memory: Samsung 8GB ECC
  • PSU: Corsair CS Series (550W)

NICs used in the videos (way overkill for home use)​

  • 4x Chelsio S320E-LP-CR

Useful Gear:​

  • 10Gb Network Interface Card
  • Direct Attach Copper Cable
  • Cat6 Cable for 10Gb (180ft max)
  • Cat6a Cable for 10Gb (330ft max)
  • 10Gb Switching
