# DnD and RPG External Resources

A collection of React components, libraries, and tools for building Dungeons & Dragons related applications.

## React Dice Components

Libraries for implementing dice rolling functionality in React applications.

- [react-dice-complete](https://github.com/AdamTyler/react-dice-complete) - 3D animated dice roller with customization options
  - [Demo](https://www.adam-tyler.com/react-dice-complete/)
- [react-dice-roll](https://www.npmjs.com/package/@nartoan/react-dice-roll) - Highly customizable dice roll component with TypeScript support
  - [Live Demo](https://nartoan.github.io/react-dice-roll/)
- [modern-react-dice-roll](https://github.com/Adarsh1999/modern-react-dice-roll) - Modernized and customizable dice roll component
- [react-dice](https://github.com/harris-jacob/react-dice) - Roll 3d dice with smooth animations

## React Component Libraries for D&D

Pre-built React component libraries specifically designed for D&D 5th Edition.

- [dungeons-and-components](https://github.com/DMYetti/dungeons-and-components) - React component library for D&D 5e
  - [Storybook Demo](https://dmyetti.github.io/dungeons-and-components/)
- [dnd-character-sheets](https://github.com/DarylBuckle/dnd-character-sheets) - D&D 5e character sheets in React
  - [Live Site](https://dnd5esheets.com/)
- [DnD-Nearby](https://github.com/LukeSlater02/DnD-Nearby) - Full character management with spellbook and tavern features
- [dnd-charactersheet](https://github.com/DanishKodeMonkey/dnd-charactersheet) - Digital character sheet manager
