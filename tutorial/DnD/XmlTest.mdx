# XmlChar Test Page

This page demonstrates the `XmlChar` component with different display modes and configuration options.

import XmlChar from '@site/src/components/XmlChar/XmlChar';

## Display: Small

The small display mode shows a compact card with essential character information: name, class, race, armor class, hit points, speed, and ability scores.

### Single Character

```mdx
<XmlChar filename="tanadirian.xml" display="small" />
```

<XmlChar filename="tanadirian.xml" display="small" />

### Multiple Characters (Default Flow)

Small cards flow naturally side by side without any wrapper. This is the default behavior.

```mdx
<XmlChar filename="murky.xml" display="small" />
<XmlChar filename="malfunction.xml" display="small" />
```

<XmlChar filename="murky.xml" display="small" />
<XmlChar filename="malfunction.xml" display="small" />

### Multiple Characters (Centered)

To center multiple small cards, wrap them in a container with `textAlign: 'center'`.

```mdx
<div style={{ textAlign: 'center' }}>
  <XmlChar filename="delizan.xml" display="small" />
  <XmlChar filename="dorian.xml" display="small" />
  <XmlChar filename="dougal.xml" display="small" />
</div>
```

<div style={{ textAlign: 'center' }}>
  <XmlChar filename="delizan.xml" display="small" />
  <XmlChar filename="dorian.xml" display="small" />
  <XmlChar filename="dougal.xml" display="small" />
</div>

### Error Handling

The component displays clear error messages when issues occur.

**File not found:**

```mdx
<XmlChar filename="fail.xml" display="small" />
```

<XmlChar filename="fail.xml" display="small" />

**Missing filename:**

```mdx

```

## Display: Medium

## Display: Large

The large display mode shows the complete character sheet, including a portrait placeholder and feats list.

### Without Portrait

When no portrait image is available or specified, a placeholder icon is displayed.

```mdx
<XmlChar filename="tanadirian.xml" image="dontexists.jpg" display="large" />
```

<XmlChar filename="tanadirian.xml" image="dontexists.jpg" display="large" />

## Display: Large with Portrait

You can specify a custom portrait image using the `image` prop. Supported formats are JPG, PNG, and SVG.

### Milo

```mdx
<XmlChar
    filename="milo.xml"
    display="large"
    image="milo.jpg"
/>
```
