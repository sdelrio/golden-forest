# /new-component <ComponentName>
Scaffold a new React component following the project's multi-file convention.

## Instructions
1. Ask for the `<ComponentName>` (PascalCase).
2. Create directory `src/components/<ComponentName>/`.
3. Scaffold the following files:
   - `<ComponentName>.js`: React 19 function component boilerplate.
   - `<ComponentName>.module.css`: Scoped CSS with a `.container` class.
   - `utils.js`: Helper functions.
4. Follow the import order and styling rules defined in `AGENTS.md`.
