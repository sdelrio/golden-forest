# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

Only the latest deployed version of [lorien.cloud](https://lorien.cloud) receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it through [GitHub Security Advisories](https://github.com/sdelrio/golden-forest/security/advisories/new).

**Do not open a public issue for security vulnerabilities.**

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response

- **Acknowledgment**: Within 72 hours
- **Triage**: Within 1 week
- **Resolution**: Depends on severity and complexity

## Scope

**In scope:**
- lorien.cloud Docusaurus site
- Custom React components (`src/components/`)
- Build pipeline and dependencies
- MDX content processing

**Out of scope:**
- Third-party services (Algolia, hosting provider)
- Social engineering attacks
- Denial of service

## Disclosure

We follow coordinated disclosure. Once a fix is deployed, we will publish a security advisory through GitHub.
